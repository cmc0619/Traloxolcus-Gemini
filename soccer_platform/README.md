# Soccer Platform (Cloud Server)

This component runs on a VPS (Virtual Private Server).

## Responsibilities

1. **Hosting**: Stores the processed video files and event logs.
2. **Database**: PostgreSQL database for complex queries ("Show me goals").
3. **Web UI**: The user-facing portal for coaches and players to watch and analyze games.

## Stack

* **Backend**: FastAPI
* **Database**: PostgreSQL
* **Frontend**: Modern Web Framework (React/Vue/Vanilla)
